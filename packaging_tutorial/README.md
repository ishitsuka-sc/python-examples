# mypackage

`mypackage` は、YAML 形式の設定ファイルを用いて挨拶メッセージを生成するシンプルな Python パッケージです。
一部のパラメータ（例: `key`）はユーザ側で必ず設定する必要があります。

---

## インストール方法

```bash
git clone https://example.com/your-repo/mypackage.git
cd mypackage
pip install -e .


## 設定ファイルの準備

ホームディレクトリに ~/.mypackage/conf.yaml を作成してください。

```
key: "YOUR_SECRET_KEY"
```
